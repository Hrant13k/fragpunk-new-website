# FragPunk Fan Website

A modern fan-made website for the game FragPunk.

This project is a redesign concept of the official website with a focus on:
- better user experience
- faster performance
- cleaner modern UI

## Features

- Home page
- Lancers page
- Weapons page
- Skins gallery
- News section

## Tech Stack

- HTML
- CSS
- JavaScript

## Assets

Images are organized in separate folders:

- `/lancers` – character images
- `/weapons` – weapon images
- `/skins` – skins and cosmetics
- `/site-assets` – UI and background images

## Purpose

This project was created for practice and portfolio purposes.
